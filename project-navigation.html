Accessible Navigation Menu

This project demonstrates a simple, accessible navigation menu built with semantic HTML and keyboard-friendly structure. The goal was to create a navigation system that is easy to use, screen‑reader friendly, and consistent across all pages of the site.

Key Features

  • Semantic <nav> element for assistive technologies
  • Clear, descriptive link text
  • High-contrast link styling
  • Keyboard-accessible tab order
  • Consistent placement across all pages

What I Learned

I learned how to structure navigation in a way that supports accessibility, improves usability, and creates a consistent experience across multiple pages. I also practiced using semantic HTML to help screen readers understand page structure.

Accessible Contact Form

This project demonstrates a fully accessible contact form built with semantic HTML, proper labels, keyboard-friendly structure, and clean, readable layout. The goal was to create a form that works for all users, including those using assistive technologies.

Key Features

  • Semantic HTML structure
  • Properly associated labels and inputs
  • Keyboard-accessible navigation
  • High-contrast button styling
  • Clear focus states for interactive elements

What I Learned

While building this form, I practiced accessible markup, improved my understanding of form semantics, and learned how to structure content so it works well for screen readers and keyboard users.
